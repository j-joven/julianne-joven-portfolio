# Julianne Joven – Portfolio

Mobile-first portfolio site showcasing selected graphic design projects.

## Pages
- Home: `index.html`
- Works: `works.html`
- About: `about.html`
- Project detail pages: `/projects/*`

## Tech
- Semantic HTML5
- Responsive CSS (Grid/Flexbox, custom properties)
- Vanilla JS (mobile nav toggle)

## Run locally
Recommended:
- VS Code extension: Live Server
- Right click `index.html` → "Open with Live Server"

## Performance notes
- Images use lazy loading below the fold
- Width/height set on images to reduce layout shift
- Minified CSS/JS included for smaller payloads
